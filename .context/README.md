# .context

Generated by Athena. This directory is the single source of truth for the
project's captured context.

- `inception-standard.md` — the 12 fields Athena extracts, with completeness rules.
- `inception.json` — the live values (Athena rewrites this per field, commits each update).
- `sources/<run_id>/` — the raw documents that seeded a given extraction run.

Do not hand-edit `inception.json` while an extraction is running — Athena
commits per field and a manual edit will be overwritten.
